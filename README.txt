Счетчик посещений v0.1

Считает количество посещенй конкретного пользователя на сайте. 
Запускается через /count?id=... и /get_count?id=... на локальном сервере.
В /count?id=... выдает картинку, занося в базу по этому id посещение.
В /get_count?id=... выдает json с количеством посещений данного айди.
