Счетчик посещений v0.2

Автор задачи: Львов Ярослав.

Описание:
	Считает количество всех посещений конкретной страницы на сайте.
	Также, умеет считывать уникальных пользователей.(WIP)
Тесты:
    Присутствуют тестовые странички по адрессной строке ../form(WIP)
	Тесты присутствуют. Расположены в \flask\tests\unit_test.py
Подробности:
	Запускается через /count?id=... и /get_count?id=... на локальном сервере.
	В /count?id=... выдает картинку, занося в общую таблицу базы и в
	таблицу уникальных посетителей.
	В /get_count?id=... выдает json с количеством общих посещений и
	уникальных за день, месяц, все время(WIP)
