Счетчик посещений v0.5

Автор задачи: Львов Ярослав.


Запуск:
    Запуск через консоль -  python3 start.py
Описание:
	Считает количество всех и уникальных посещений конкретной
	страницы на сайте.
Состав:
    Cам счетчик и статистика: start.py
    Sqlite запросы: sqlite_manager.py
    Тесты: flask_tests/
    Изображение - 1х1 прозрачный пиксель: static/img/
    Рабочая база данных: BD.db
    HTML странички: templates/
Тесты:
	Юнит тесты присутствуют. Расположены в flask/tests/unit_test.py
	Coverage: 60%
Подробности:
	Посмотреть работоспособность счетчика можно
	через /count?id=... и /get_count?id=... на локальном сервере.
	В /count?id=... выдает картинку, занося в общую таблицу базы и в
	таблицу уникальных посетителей.
	В /get_count?id=... выдает json с количеством общих посещений и
	уникальных за день, неделю, месяц.
    Чтобы посмотреть статистику по странам на определенной странице нужно
    сделать такой запрос /get_country?id=...&country=1. В таком случае
    выдается dict {country : all_time_visits}
